# Ten_syoku_bot

这是一个用于自动化抓取Recruit Agent数据并进行结构化清洗的工具。
项目集成了网页自动化脚本与数据处理，旨在提高JD的获取与分析效率。

Recruit Agent の求人情報を自動取得し、構造化データにクレンジングするためのツールです。
Webオートメーション（スクレイピング）とデータ処理を統合し、求人票の収集と分析効率を向上させることを目的としています。
