# Ten_syoku_bot

这是一个用于自动化抓取Recruit Agent数据并进行结构化清洗的工具。
项目集成了网页自动化脚本与数据处理，旨在提高JD的获取与分析效率。


### 第一步：配置脚本
1. 打开 `JDbot_ver3.py`，根据抓取需求修改以下信息并保存：
2. 修改 【搜索结果】
3. 修改 【更新日期】

### 第二步：执行抓取
1. 运行主程序：`python JDbot_ver3.py`
2. 在自动弹出的网页窗口中完成 **账号密码登录**。
3. 回到 CMD界面，**按回车键**开始抓取。
4. **手动中断**：当观察到CMD开始大量出现“跳过”提示时，可中断运行。

### 第三步：数据清洗
1. 打开 `data-cleansing_ver3.ipynb`。
2. 运行所有Cell，自动处理原始数据。

### 第四步：结果导出
1. 打开 `jd_data_structured.csv` 。
2. 使用 **Excel** 打开该文件。
